Webware for Python 0.3 Release Notes

Release Notes
  • Version 0.3 was released on 6/6/2000.
  • There is an improved WebKit 0.3 with new features, refinements and bugfixes.
  • There is a new PSP component (aka Python Server Pages) which is a WebKit plug-in that enables you to create server pages that freely mix HTML and Python in the same fashion as other popular server technologies such as ASP, PHP and JSP.
  • The project is now fully hosted on SourceForge at http://webware.sourceforge.net including the CVS source code repository. You can anonymously checkout a snapshot of the repository if you want the latest, in-development version.
  • Webware is now friendlier to Windows including both code and documentation. We now have developers using both Windows and UNIX/Linux.
  • As usual, the bulk of the documentation and release notes is with the components themselves. Please use the links above.
