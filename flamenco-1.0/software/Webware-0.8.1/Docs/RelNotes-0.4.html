Webware for Python 0.4 Release Notes

Release Notes
  • Version 0.4 was released on 9/28/2000.
  • Webware has found use in the real world resulting in plenty of excellent feedback, refinements and bugfixes.
  • There is a new install.py script in the main Webware directory that is required to be run.
  • The install script, among other things, generates HTML class hierarchies, file summaries and syntax-highlighted source as part of the documentation.
  • A comprehensive installation guide for WebKit.
  • More integration with other web technologies like mod_python, PyWX/AOLserver, etc.
  • Better behavior on Windows with no known Windows problems.
  • MiscUtils has been expanded on, and documented for the first time.
  • There is an improved WebKit 0.4.
  • There is an improved PSP 0.2.
  • As usual, the bulk of the documentation and release notes is with the components themselves. Please use the links above.
