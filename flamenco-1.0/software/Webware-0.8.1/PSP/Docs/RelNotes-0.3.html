PSP 0.3 Release Notes

Version 0.3
Webware for Python 0.4

Release Notes
  • Version 0.3 was released on 9/28/2000.
  • Version 0.3 represents a stabilization of the basic syntax and raw capabilities of the PSP module for WebKit. All of the basic functionality is there and works as expected. The only major addition for this release is a new, optional syntax for handling Python blocks that need to include straight HTML (curly braces).
